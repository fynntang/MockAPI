## 📝 Description

MockAPI is a lightweight local API mock server for developers who want to move fast.

### Features

🎨 **Web UI** — Dark theme interface to manage routes visually  
📜 **JavaScript Scripts** — Dynamic responses with full scripting power  
📥 **Swagger Import** — Auto-generate mocks from OpenAPI specs  
🔌 **WebSocket & GraphQL** — Mock modern APIs  
🔄 **Proxy Mode** — Forward unmatched requests to real backend  
⚡ **Zero Dependencies** — Single Go binary, no Node.js or database  

### Perfect For

• Frontend developers waiting for backend APIs  
• Testing edge cases and error responses  
• Prototyping and rapid development  
• CI/CD pipelines and automated testing  

**Open source and free. Built with Go.**
