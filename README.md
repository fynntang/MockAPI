# MockAPI Project

## Quick Start

    # Start server
    mockapi serve

## Project Structure

- routes/ - Route configuration files
- scripts/ - JavaScript script files
- protos/ - Protocol buffer definitions
- mockapi.json - Main configuration file

## Add Your First Route

1. Open Web UI: http://localhost:8088
2. Click "+ Add Route"
3. Configure method, path, and response
4. Test: curl http://localhost:8088/mock/your-path

Learn more: https://github.com/fynntang/MockAPI
